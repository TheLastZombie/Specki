## [Running on your Machine](../../wiki/Run-ðŸ‡ºðŸ‡¸)

<details><summary>Manually</summary>
<p>

```
git clone https://github.com/TheLastZombie/Specki
cd Specki
npm install
```

Replace the environment variables in `index.js` and all commands.

```
npm start
```

</p>
</details>

<details><summary>dotenv</summary>
<p>

```
git clone https://github.com/TheLastZombie/Specki
cd Specki
npm install
```

Replace the environment variables in `.env`.

```
node -r dotenv/config index.js
```

</p>
</details>

<details><summary>Heroku</summary>
<p>

```
git clone https://github.com/TheLastZombie/Specki
cd Specki
npm install
```

Replace the environment variables in `.env`.

```
heroku local
```

</p>
</details>
